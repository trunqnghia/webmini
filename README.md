
# Mini-project 1: Trang HTML chuẩn đầu tiên

Dự án này hoàn thành đầy đủ các yêu cầu về cấu trúc tài liệu HTML, giải thích DOCTYPE, html, head, body và tổ chức thư mục assets bài bản.

## mô tả phân vùng HTML theo ảnh thiết kế
Dựa trên ảnh chụp thiết kế UI/page, cấu trúc file index.html được phân vùng chi tiết như sau:

1. **Phân vùng Đầu trang (`<header>` và `<nav>`)**: Vùng thanh công cụ điều hướng menu trên cùng của ảnh thiết kế.
2. **Phân vùng Nội dung chính (`<main>`)**: Vùng hiển thị nội dung bài học ở cột bên trái của ảnh thiết kế, chứa các thẻ section giải thích DOCTYPE, html, head, body.
3. **Phân vùng Thanh bên (`<aside>`)**: Vùng cột bên phải trên ảnh thiết kế, dùng hiển thị sơ đồ phân vùng trực quan.
4. **Phân vùng Chân trang (`<footer>`)**: Vùng tối dưới cùng chứa thông tin bản quyền theo ảnh thiết kế.
