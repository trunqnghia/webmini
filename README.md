# Mini-Project 1: Trang HTML Chuẩn Đầu Tiên

Dự án này là trang tổng hợp kiến thức nền tảng về cấu trúc tài liệu HTML5, giải thích tường tận ý nghĩa các thẻ cấu hình cốt lõi và thực hiện phân chia vùng Layout dựa trên bản vẽ thiết kế UI.

## 🛠 Công nghệ sử dụng
- **HTML5** (Sử dụng các thẻ Semantic ngữ nghĩa như `header`, `nav`, `main`, `section`, `aside`, `footer`)
- **CSS3** (Bố cục cấu trúc bằng Grid System & Flexbox, hỗ trợ Responsive đầy đủ trên các thiết bị)
- **JavaScript ES6** (Xử lý vòng đời tải trang)

## 📁 Cấu trúc cây thư mục dự án
- `index.html`: Trang đích chính hiển thị nội dung và phân tách mã nguồn rõ ràng.
- `css/style.css`: Quy định định dạng thẩm mỹ, màu sắc, phân chia lưới giao diện.
- `js/script.js`: Xử lý logic nghiệp vụ và tương tác hệ thống.
- `assets/ui-design.png`: Ảnh chụp phân vùng thiết kế UI gốc được đối chiếu để cắt code HTML.
