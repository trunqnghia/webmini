Mini Project HTML5

Project mô tả cấu trúc tài liệu HTML5 bằng giao diện trực quan, semantic HTML và DOM Tree hiện đại.

Khám phá Project

Giới thiệu

Đây là mini-project môn HTML/CSS nhằm giúp sinh viên hiểu rõ:

DOCTYPE

Khai báo chuẩn HTML5 cho trình duyệt.

DOM Tree

Minh họa cấu trúc lồng nhau của HTML.

Semantic HTML

Sử dụng header, main, section, footer...

Responsive

Tương thích trên desktop và mobile.
